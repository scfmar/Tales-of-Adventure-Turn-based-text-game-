Required software:
python 3.9.6

Before accessing the application:
access the application folder through CMD and create a virtual environment folder to store the following libraries in "requirements.txt"

To access the application:
>Enter CMD
>access the application folder
>Activate the virtual_environment
cd (name of virtual folder)
cd scripts
Activate

>go back to the application folder
>type in "python .\Play.py" to start the game
